# Betreuungsrecht

**Skills nach der Reform 2023 (§§ 1814 ff. BGB).** Für berufliche Betreuer:innen und Betreuungsbehörden.

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `jahresbericht-1863-bgb` | Jahresbericht ans Betreuungsgericht | § 1863 BGB, BtOG |
| `vermoegensverzeichnis` *(geplant)* | Vermögensverzeichnis bei Beginn und Rechnungslegung | §§ 1835, 1865 BGB |
| `genehmigungspflicht` *(geplant)* | Genehmigungspflicht-Prüfung (Vermögenssorge) | §§ 1848 ff., 1831, 1832 BGB |
| `vorsorgevollmacht` *(geplant)* | Abgrenzung zur Vorsorgevollmacht | §§ 1820 BGB |
| `betreuungsbestellung` *(geplant)* | Bestellung durch Betreuungsgericht | §§ 1814 ff. BGB |
