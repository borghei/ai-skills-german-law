# Erbrecht

**Skills für die Fachanwaltschaft Erbrecht.**

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `testament-pruefung` | Formelle und materielle Wirksamkeit eigenhändiges/notarielles Testament | §§ 2247 ff. BGB, § 2229 BGB |
| `pflichtteil` *(geplant)* | Pflichtteilsanspruch und Ergänzungsanspruch | §§ 2303 ff., 2325 BGB |
| `erbvertrag` *(geplant)* | Erbvertrag, Bindungswirkung, Aufhebung | §§ 2274 ff. BGB |
| `erbschein` *(geplant)* | Antrag und Verfahren beim Nachlassgericht | §§ 2353 ff. BGB, §§ 342 ff. FamFG |
| `erbengemeinschaft` *(geplant)* | Auseinandersetzung, Teilungsversteigerung | §§ 2032 ff. BGB |
| `erbschaftsteuer` *(geplant)* | Freibeträge, Bewertung, Anzeige | ErbStG |
