# Gewerblicher Rechtsschutz

**Skills für die Fachanwaltschaft Gewerblicher Rechtsschutz (Markenrecht, Patentrecht, Designrecht, UWG).**

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `markenrechtsverletzung-abmahnung` | Außergerichtliche Abmahnung und Unterlassungsanspruch wegen Markenrechtsverletzung | §§ 14, 15 MarkenG, §§ 8 ff. UWG |
| `patentverletzung` *(geplant)* | Patentverletzungsklage, Auskunfts- und Schadensersatzanspruch | §§ 139 ff. PatG |
| `designverletzung` *(geplant)* | Design-/Geschmacksmusterverletzung | §§ 38 ff. DesignG |
| `uwg-irreführung` *(geplant)* | Irreführende Werbung und Mitbewerber-Abmahnung | §§ 5, 5a UWG |
| `markenanmeldung` *(geplant)* | Anmeldung beim DPMA / EUIPO | MarkenG, UMV |
