# IT-Recht

**Skills für die IT-rechtliche Praxis (Fachanwaltschaft).**

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `it-vertragspruefung` | Prüfung typischer IT-Verträge (SaaS, Cloud, Outsourcing) | §§ 631, 535, 280 ff. BGB; § 28 DSGVO |
| `softwarelizenz-pruefung` *(geplant)* | Lizenzmodelle und Reichweite, Open-Source-Compliance | UrhG, BSD/GPL/MIT |
| `cloud-avv` *(geplant)* | DSGVO-konformer Auftragsverarbeitungsvertrag für Cloud-Dienste | Art. 28 DSGVO |
| `evb-it` *(geplant)* | Öffentliche Beschaffung mit EVB-IT-Mustern | EVB-IT, VOL/A |
| `ki-vertragspruefung` *(geplant)* | Verträge mit KI-Anbietern (Verzahnung mit ki-vo-compliance) | KI-VO, § 43e BRAO |
