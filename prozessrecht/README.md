# Prozessrecht

**Skills für die zivilprozessuale Praxis.**

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `klageschrift-zivilprozess` | Strukturierte Klageschrift §§ 253 ff. ZPO inklusive Streitgegenstand, Sachvortrag und Beweisantritt | §§ 253, 261 ZPO |
| `klageerwiderung` *(geplant)* | Klageerwiderung mit Sachvortrags-Replik | § 277 ZPO |
| `beweisantrag` *(geplant)* | Beweisanträge: Zeuge, SV-Gutachten, Urkunde, Augenschein | §§ 373 ff. ZPO |
| `rechtsmittel-berufung` *(geplant)* | Berufung §§ 511 ff. ZPO | §§ 511 ff. ZPO |
| `einstweiliger-rechtsschutz` *(geplant)* | Arrest und einstweilige Verfügung | §§ 916 ff. ZPO |
