How it works

Sourced citations and a paper trail — not just prompt templates

Researcher → Drafter → Reviewer

Three sub-agents per skill: Quellensuche, Entwurf im Gutachtenstil, Risiko- und Berufsrechts-Check. The reviewer sub-agent blocks unsourced citations.

Linked primary sources

Every § citation links to gesetze-im-internet.de, every EU regulation to EUR-Lex. Case law explicitly marked verified or [unverifiziert, prüfen].

Multi-provider, one source

One canonical SKILL.md per skill. The router script regenerates Claude Code / Gemini Gem / OpenAI Custom GPT adapters on demand. No drift between providers.

Dated verification log

Every source-verification pass is logged with date, source, and corrections. Last pass 2026-05-21. Recent fixes: LkSG § 24 Bußgeldhöhe, KSchG § 23 Leiharbeitnehmer, CSRD Omnibus.

370+ assertions, CI-tested

Every skill ships with a structured test.md with must-cite, must-appear, and must-flag assertions. scripts/eval.py runs on every push: 28/28 passing.

§ 203 + DSGVO + KI-VO scaffolding

PII redaction script, gateway-routing walkthrough for Langdock/deepset, compliance checklist for § 203 StGB, DSGVO Art. 28-44, KI-VO Art. 6/26/50, Schrems II.

How much you can rely on it

Production-grade today

Plugin manifests validate. Statute citations link to gesetze-im-internet.de and EUR-Lex. Methodology textbook-correct (Gutachtenstil, Anspruchsgrundlagen-Reihenfolge, BGH/Beck-Zitierweise). Multi-provider parity via scripts/route_provider.py.

In active improvement

Every BAG / BGH / EuGH citation carries [unverifiziert, prüfen] until a PR replaces it with a sourced URL. Legal-accuracy eval (currently structural-only) is next. Rechtsanwalt review per area is the third axis.

Hard limits

No real Mandatsdaten without a § 203-compliant gateway (Anthropic / Google / OpenAI do not sign Verschwiegenheitserklärungen for German lawyers). Every output is a draft for review by a licensed Rechtsanwalt under § 43a BRAO and § 2 BORA.

Install in seconds

Three paths, depending on which LLM you use

Claude Code (native)

claude>/plugin marketplace add borghei/AI-Skills-German-Law
claude>/plugin install arbeitsrecht

Then call any skill: /arbeitsrecht:kuendigungs-pruefung

Gemini Gems or OpenAI Custom GPT

$git clone https://github.com/borghei/AI-Skills-German-Law.git
$python scripts/route_provider.py --provider gemini --skill arbeitsrecht/kuendigungs-pruefung

Renders the Gem instructions or Custom GPT system prompt. Paste into gemini.google.com or chatgpt.com.
